## 一些日志
- 博客模板: vitepress  
- vitepress [官方网址]: (https://vitepress.dev/reference/)
- 博客搭建工具: npm, cnpm, markdown
- vitepress优势: 
  - vite工具搭建,
  - 热更新快, 
  - 搭建简单,
  - 自带代码高亮, 
  - 日夜模式, 
  - 侧边栏, 
  - 主题配置.

## 博客启动
> npm run docs:dev
## 博客目录
- docs 项目目录
  - .vitepress 配置文件
  - index.md 首页
